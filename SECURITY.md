# Security Policy

## Reporting a vulnerability

If you find a security issue in OpenShard, please do not open a public 
GitHub issue.

Instead, report it privately by emailing: **michaelobasa2@gmail.com**

Include:
- A description of the issue
- Steps to reproduce it
- Any relevant context about impact

We'll respond as quickly as possible and work with you on a fix before 
any public disclosure.

## Scope

This covers the OpenShard CLI and routing engine. Issues related to 
third-party model providers should be reported directly to those providers.
